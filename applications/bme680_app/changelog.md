## v2.0
- Initial public release
- Real-time temperature, humidity, pressure, and gas resistance readings
- Calculated IAQ score (0-500) with visual progress bar
- Comfort index combining temperature and humidity assessment
- Heat index (feels-like temperature) using Rothfusz regression
- Absolute humidity calculation in g/m³
- Dew point calculation
- 3 data pages: Overview, Gas & Air Quality, Comfort
- Configurable alarms for IAQ, temperature, and humidity thresholds
- Alarm outputs: LED (Off/Red/Green/Orange), vibration, and sound
- SD card CSV logging with date/time stamps, one file per session
- Configurable settings: I2C address, temperature unit (C/F), reading interval, heater temperature
- Auto-retry sensor detection (never stops trying)
- Dual I2C address support (0x76 / 0x77)
- Settings and alarms accessible via long press OK overlay menu
