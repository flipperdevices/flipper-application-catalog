# MultiTimer

Timer app for Flipper Zero with saved timer presets, a clock screen, and optional WeActStudio 4.2" e-paper output.

## Requirements

- Flipper Zero with official firmware (Release or Release Candidate)
- No external hardware required for core timer features
- Optional WeAct 4.2" black/white e-paper module for external display output
